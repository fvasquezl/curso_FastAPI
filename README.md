## Instructions to run this API

### Clone the repository
```
$ cd curso_FastAPI
$ pipenv install -r requirements.txt
$ pipenv shell 
$ uvicorn main:app --reload --port 5000 --host 0.0.0.0

```
### Open the browser
```
localhost:5000/docs
```
## Libraries
```
Api: fastapi
Tokens: PyJWT
Orm: SQLAlchemy
WebServer: uvicorn

```
